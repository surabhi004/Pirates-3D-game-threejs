## RUN THE GAME

1. npm install

2. npm run dev

3. Go to localhost:3000 preferably on google chrome

## PLAY

```

Key Bindings

|   Key    |       Description        |
|----------|--------------------------|
|    W     |     move forward         |
|    S     |     move backward        |
|    D     |     move right           |
|    A     |     move left            |
|  Space   |   Shoot cannonballs      |
|  Shift   |  switch to bird eye view |

```
